# TicketOn

TicketOn es una aplicación desarrollada en Python para la gestión de eventos y venta de entradas.

## Objetivo

Desarrollar una plataforma que permita:

- Gestionar usuarios.
- Gestionar eventos.
- Vender entradas.
- Validar tickets.
- Generar reportes.

## Tecnologías

- Python 3.12.6
- SQLite
- Git
- Visual Studio Code
